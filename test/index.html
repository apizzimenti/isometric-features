This game toolkit is robust, with many available features and integrations. Make sure to check the documentation out.

Movement

The main character is moved around by the arrow keys: ←, ↑, →, and ↓.

Items

Items aren't critical and definitely don't have to be in your game, but they are a good addition to any. Building a working inventory for any game is tough, let alone building it so that your items can interact in a 3d world. The Inventory class allows for a fully-functional fillable inventory with custom items. Provided (as part of the toolkit) is an Item class that can be extended.

Importance

This game system is important for those that want to create highly interactive games with the Phaser platform, but want the isometric library to be a part of their system.
